Nonogram Solver


I made a program to automatically solve nonograms on Puzzle Nonograms. The program takes in the nonogram dimensions, the html for the hints, the location of the top left corner, and the location for the bottom right corner. Then the program extracts the hints from the html, generates possible permutations, and eliminates them iteratively. Finally, the program uses the pixel locations and the pyautogui library to automatically click the cells.

Fluid Flow Simulation


In my differential equations class, I programmed a MATLAB simulation of fluid flow using Darcy's law. The method involves splitting up the flow regions into cells and assuming the cells have uniform material properties within them.
